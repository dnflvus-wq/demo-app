## FINAL 실험의 특히 우수한 TC

### TC-9: 수정 모드 진입 및 저장 (가장 완성도 높은 TC)
```
testStep:
1. /todo 페이지 접속
2. 목록의 항목에서 '수정' 버튼 선택
3. 제목/설명/우선순위가 폼에 로드되는지 확인  ← 기존값 로드 검증!
4. 제목을 '수정된 제목'으로 변경
5. [저장] 버튼 선택

expectedResult:
- 수정 폼에 기존 제목/설명/우선순위가 로드되어 출력  ← 수정 모드 UI 검증
- 저장 후 목록의 항목이 '수정된 제목'으로 변경되어 출력
- 수정 완료 시 토스트 또는 완료 메시지 출력  ← 토스트 검증
```
→ **TODO-005 + TODO-020을 하나의 TC에서 동시 커버!** Gold Standard 기준 2개 TC의 내용을 잘 통합.

### TC-4 + TC-5: 경계값 완벽 분리
- TC-4: 50자 입력 → 정상 추가 (경계값 직전)
- TC-5: 51자 입력 → 제한/오류 (경계값 직후)
→ **경계값 직전/직후 분리 테스트가 QA 실무 표준과 일치!**
